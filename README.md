# Surebet Finder

## Calcul

### Find a surebet

- Team 1 : 2.6
- Nul : 3.3
- Team 2 : 3.3


- Rate Team 1 = 1 / 2.6 = **0.384**
- Rate Nul = 1 / 3.3 = **0.303**
- Rate Team 2 = 1 / 3.3 = **0.303**


- Total = Rate Team 1 + Rate Nul + Rate Team 2
- Total = 0.384 + 0.303 + 0.303
- Total = 0.99


- **Total < 1 = Surebet**

### Calcul Bet

```Sum / ( Total * Odd )```

Example with 100 € :
- Team 1 = 100 / (0.99 * 2.6) = 38.8 €
- Team 1 = 100 / (0.99 * 3.3) = 30.6 €
- Team 1 = 100 / (0.99 * 3.3) = 30.6 €
